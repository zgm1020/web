# 我的博客（可网络访问）

这是一个纯 HTML/CSS/JS 的静态博客。

## 本地启动

```bash
npm start
```

默认会监听：

- Host: `0.0.0.0`
- Port: `8080`

> 监听 `0.0.0.0` 的意思是：同一局域网中的其他设备可以通过你的机器 IP 访问。

## 在局域网中访问

1. 启动服务：`npm start`
2. 查看本机 IP（例如 `192.168.1.23`）
3. 在其他设备浏览器打开：`http://192.168.1.23:8080`

## 在云服务器上公网访问

1. 把项目部署到云服务器
2. 启动服务：`npm start`
3. 放行服务器安全组/防火墙的 `8080` 端口
4. 用公网 IP 访问：`http://<你的公网IP>:8080`

## 自定义端口

```bash
PORT=3000 npm start
```

然后访问：`http://<IP>:3000`

## 文件结构

- `index.html`：页面结构
- `styles.css`：样式
- `app.js`：文章渲染、搜索、主题切换
- `server.js`：静态文件服务（网络访问入口）
